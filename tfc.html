Team Fortress Classic Colors

A quick side section relating to Team Fortress Classic. If you've played Team Fortress 2, you may be familiar with how that game handles team colors, which is to say, it uses separate textures for its different teams. TFC does not do this. Instead it uses preset colors for every team built directly into the topcolor and bottomcolor system.

For most classes, I will provide all of the different color values here on this page. However, I should note that the colors used change depending on if you are using the old model or the new model. Old models only use one shared set of colors that is used for each team. New models, on the other hand, have unique values for each class.
